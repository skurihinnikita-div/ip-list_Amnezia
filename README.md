# 🛡 AmneziaVPN: списки CIDR для Split Tunneling

Готовые наборы IP-сетей (CIDR) для настройки **только нужных сайтов через VPN** в AmneziaVPN.  
Помогают обходить блокировки, не нагружая канал и процессор лишним трафиком.

## 📦 Что внутри

| Файл | Количество сетей | Покрытие |
|------|----------------|-----------|
| `compact.json` | ~96 | Только топ‑20 сервисов (Telegram, YouTube, Google, Meta, Discord, Netflix, Spotify, AWS, Cloudflare, Hetzner) |
| `medium.json` | ~365 | Почти все зарубежные CDN, облака, хостинги, соцсети (расширенный вариант) |
| `full.json` | ~1600+ | Весь IPv4 (тормозит, не рекомендуется) |

## 🚀 Быстрый старт

1. Скачайте нужный JSON‑файл.
2. В AmneziaVPN: **Настройки → Split Tunneling → Only specified sites go through VPN**.
3. Нажмите **Import** и выберите файл.
4. Переподключите VPN.

> ✅ Списки содержат только CIDR‑сети (IP с масками), поэтому работают быстро и без DNS‑прокси.

## 🧠 Рекомендация

- Начните с `compact.json` — его хватит 99% пользователей.
- Если какой‑то сайт не открывается — добавьте его IP‑сеть вручную (через `nslookup` + `whois`).
- `medium.json` — для максимального покрытия без сильной нагрузки на CPU.
